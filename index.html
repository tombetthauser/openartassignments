Open Art Assignments

This page is a repository of links to art assignment pdf's and google docs that are intended for adjunct arts instructors to freely view, download, change and use in their courses. Please also feel free to share this page with any adjunct instructors that you think might be able to use these to save time or inspire new course content.

To edit any of these assignments, click on the google-docs link, find the google docs dropdown menu labelled File and choose Make a Copy. You may have to sign in to google docs or make an account if you don't have one.

If you have any interest in uploading your own assignments to the page for open use by other instructors we'd be happy to add it. Just email a pdf, google doc link, word file etc to the email below with the subject line, Open Art Assignment.
